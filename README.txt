หลังอัปโหลดไฟล์:
1) เปิด login.html → เข้าระบบ
2) ไปที่ "ตั้งค่า" → กด "ล้าง Cache / โหลดใหม่" หนึ่งครั้ง เพื่อให้ SW โหลดรุ่นล่าสุด
3) ตั้งค่าโควตาเริ่มต้นตามต้องการ (เช่น 30)
4) ตั้งค่าเงินเดือนเพื่อคำนวณอัตราต่อชั่วโมง (ถ้ายังไม่ได้ตั้ง)
